## What changed in v0.2 vs v0.1

**Eval integration:**
- All `[.]` placeholders in Tables 2 and 3 replaced with real bootstrap numbers
- New "Bootstrap reproduction" subsection (§6.2) honestly frames what was actually run: 20 patients / 207 notes / 100 queries, MiniLM-L6-v2 embeddings, deterministic seed 42
- New Table 2 (per-category R@5) showing the lookup-query gap that drives the headline finding
- Three findings reworded with the actual deltas (40.9pp drop for read-only, 67.4% leak, BDP R@1 within 0.001 of Raw)
- Ablation result (Table 3) now leads with **HMAC determinism as the largest contributor** (−0.301 vs BDP, collapsing to *exactly* symmetric-redaction recall), which is the empirical core of the paper
- Bootstrap-scale caveat added as the first item in Limitations
- Data & Code Availability now lists the actual eval files (`bdp.py`, `corpus.py`, `pipelines.py`, `run.py`, `ablate.py`, `smoke_test.py`) and the regeneration command
